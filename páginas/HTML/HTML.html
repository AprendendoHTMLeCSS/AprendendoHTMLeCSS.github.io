Aprendendo HTML!

Usados inicialmente para a construção de paginas web, as tags HTML hoje estão presentes nas mais diversas aplicações com intuito de incorporar conteúdos web nas mesmas. Por isso é muito importante que programadores e pessoas da área de TI aprendam ao menos o básico das tags HTML. Então vamos começar!

Atenção! todas as tags exibidas nesse conteúdo estão sendo escritas com o formato: <.tag>, porém no codigo não deve haver esse ponto, essa foi uma forma de mostras a tag sem bugar o código da pagina ;)

O que são e como funcionam as tags HTML?

Elas servem para que o navegador formate e mostre o conteúdo de forma correta. As tags são sempre acompanhadas Chevron(<>),os famosos "maior que" e "menor que". As tags principais são compostas de três partes principais: tag de abertura, conteúdo e fechamento (algumas, como a tag de pular linha <.br>, não precisam ser fechadas).

Principais tags do HTML e suas definições:

  • <.html>: Define o início e o fim do documento HTML.
  • <.head>: Contém informações sobre o documento, como o título, links para CSS e JavaScript, etc.
  • <.title>: Define o título da página, que é exibido na aba do navegador.
  • <.body>: Contém o conteúdo da página, como textos, imagens, vídeos, etc.
  • <.h1>a <.h6>: Define os títulos e subtítulos da página, com <.h1> sendo o maior e <.h6> o menor.
  • <.p>: Define um parágrafo de texto.
  • <.a>: Define um link para outra página ou recurso.
  • <.img>: Insere uma imagem na página.
  • <.ul> e <.li>: Cria uma lista com marcadores.
  • <.ol> e<.li>: Cria uma lista numerada.
  • <.div>: É como um container, cria um bloco de conteúdo que pode ser estilizado com CSS.
  • <.span>: Cria um elemento de texto que também pode ser estilizado com CSS.
  • <.table>, <.tr>, <.th>, <.td>: Cria uma tabela com linhas e colunas.
  • <.form> e <.input>: Cria um formulário para coletar informações do usuário.
  • <.select>, <.option>: Cria uma caixa de seleção com opções.
  • <.textarea>: Cria uma área de texto para o usuário inserir informações.
  • <.button>: Cria um botão clicável.
  • <.script>: Permite a inclusão de scripts JavaScript na página.

Tags Semânticas

Os elementos semânticos, ao contrário de outros elementos como o <.div>, pode comunicar o seu conteúdo no seu código, facilitando a diferenciação do resto do site no código.

  • <.article>: É considerado uma parte quase independente do site, onde apresenta um título e uma descrição. Normalmente usado em blogs, fóruns e sites de notícia.
  • <.aside>:Deixa o conteúdo em seu interior ficar ao lado do resto do conteúdo. O conteúdo do aside deve ser no mínimo, indiretamente ligado ao conteúdo que ele está ao lado.
  • <.details>:É um conteúdo que pode ser contraído e extendido pelo usuário, ao estar minimizado, apresenta apenas um pequeno título, con o conteúdo escondido até que ele seja clicado.
  • <.figcaption>:É usado como legenda de uma imagem <.figure.>
  • <.figure>:Um elemento feito pra marcar uma foto.
  • <.footer>:Usado para definir o rodapé de uma página, normalmente lá você coloca informações como copyright, autoria, documentos relacionados, fontes, etc.
  • <.main>:Utilizado para marcar o conteúdo principal de uma página.
  • <.mark>:Mostra os textos de que serão marcados/grifados, sendo colocado em destaque.
  • <.summary>:Aqui você colocará o texto que será o texto completo que fica no <.details> ao ser extendido.
